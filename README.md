# A blueprint react application with express server, babel and webpack. 
Heroku ready. 

## Available Scripts
### `yarn start`
Starts the server and makes use of the buid.

### `yarn dev`
Runs the application in developer mode.

### `yarn build`
Buils the application using webpack and babel.

### `yarn test`
react-scripts test

### `yarn eject`
react-scripts eject
